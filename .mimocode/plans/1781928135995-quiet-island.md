# План: Картинка для рекламы RawLead в TG-чатах

**Формат:** 1080×1080
**Стиль:** NEO-BRUTALIST (жёлтый #FACC15, чёрный #0A0A0A)
**Цель:** Показать 2 ключевые фичи — ИИ-отклик и персональную ленту

---

## Что показывает

| Элемент | Что communicates |
|---------|------------------|
| Лента с match % | **Персональная лента** — заказы под твой стек |
| Черновик отклика | **ИИ-отклик** — готовый текст, копируй и отправляй |
| @rawlead_bot | Куда идти — искать бота в Telegram |
| «Trial бесплатно» | Снижение порога входа |

## Детали

- **Карточки в ленте:** стилизация под сайт RawLead — чёрная тень, белый фон, зелёный match %
- **Черновик:** моноширинный шрифт, как терминал/код
- **Стрелка:** связывает ленту → отклик (показывает поток)
- **Минимум текста:** только заголовок + @rawlead_bot + trial

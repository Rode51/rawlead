**Назначение:** PNG/SVG от Designer (Wave 2, переосмысление UI) — hero, иллюстрации, иконки, mood.

**Генерация:** Recraft MCP в Cursor → `@designer` · канон: [`docs/team/common/MCP_POOL.md`](../../team/common/MCP_POOL.md) § Recraft.

**Именование:**

```
wave2-{страница}-{элемент}-{vN}.{svg|png}
```

Примеры: `wave2-mark-radar-v1.svg` · `wave2-hero-geo-corner-v1.svg` · `wave2-lenta-empty-v1.png`

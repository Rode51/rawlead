# FL Radar

---

## Что за проект и какую проблему решает

На FL.ru и Kwork каждый час появляются новые заказы — вручную обновлять вкладки и пролистывать ленту отнимает время, а подходящие проекты легко пропустить.

**FL Radar** автоматизирует поиск: сам смотрит ленты по расписанию, отбрасывает нерелевантное и присылает в Telegram только новые заказы, которые прошли фильтры — с кратким разбором нейросетью (стоит ли брать, срок, деньги, риски).

Итог: не нужно сидеть на бирже — решение об отклике принимаешь по готовой карточке в телефоне.

---

## Технический стек

Python · SQLite · requests · BeautifulSoup · OpenRouter API · Telegram Bot API

---

## Логика работы (на пальцах)

1. **Сбор** — скрипт забирает свежие карточки с FL.ru и Kwork (публичные страницы ленты).
2. **Отбор** — сверка с базой (уже видели — не шлём), фильтр по словам и бюджету.
3. **Анализ** — для подходящих заказов текст уходит в LLM (OpenRouter): вердикт, риски, черновик отклика.
4. **Доставка** — готовое сообщение приходит в Telegram; пауза и статус — кнопками в том же боте.

---

## Результат

- Радар крутится **в фоне** на ПК: опрос лент каждые 10+ минут без ручного обновления сайта.
- За один проход обрабатывает **десятки–сотни** карточек (FL — до ~90 с нескольких страниц, Kwork — лента проектов), в Telegram попадают только **новые** и прошедшие фильтр.
- **ИИ отсекает мусор** (вёрстка, легаси, мелкий бюджет) — в уведомлениях остаётся разбор «брать / мимо» с экономикой и рисками.
- **Ночью** можно поставить на паузу из Telegram — парсинг и нейросеть не работают, время на ручной мониторинг не тратится.

## Промпт (копипаст)

```
Контекст: я фриланс-исполнитель (не классический программист). Делаю через Cursor: Telegram-боты, автоматизация, парсеры открытых сайтов, Google Таблицы, простые интеграции, WordPress-правки. Силён в логистике и диспетчеризации (~4 года). Ищу заказы в Telegram-чатах и каналах, куда можно вступить по invite-ссылке и мониторить новые посты «ищу исполнителя / нужен специалист».

Задача: составить МАКСИМАЛЬНО полный практический список русскоязычных Telegram-чатов и каналов (2024–2026), где публикуются заказы/запросы на услуги, а не реклама курсов и не чаты «только вакансии в штат».

Нужны источники с высокой долей постов вида:
- «нужен бот / автоматизация / парсер / скрипт»
- «нужен сайт / WordPress / Tilda / лендинг»
- «интеграция CRM / Google Sheets / API»
- «логистика / маршруты / 1С / диспетчеризация / заявки»
- «админка / настройка сервера / хостинг» (простые задачи)
- дубли заказов с FL.ru / Kwork в виде постов в TG

Исключить (не включать в список или пометить «мусор»):
- чаты только с вакансиями (найм в компанию, оклад)
- крипто/казино/ставки/серые схемы
- накрутка, отзывы, регистрация аккаунтов
- чистый дизайн/рисование без разработки
- мёртвые чаты (нет постов 30+ дней)
- закрытые без публичной invite-ссылки (если ссылку нельзя получить — отдельная секция «нужен ручной доступ»)

Формат ответа — таблица Markdown:

| № | Название | @username или ссылка t.me/... | Тип (чат/канал) | Тематика | Активность (высокая/средняя/низкая) | Подходит под Cursor-исполнителя (да/частично/нет) | Примечание |

Сгруппируй секциями:
1. Фриланс-биржи и агрегаторы заказов в TG
2. IT / разработка / боты / автоматизация
3. No-code / WordPress / Tilda / сайты
4. Логистика / транспорт / 1С / бизнес-процессы
5. Маркетплейсы / e-commerce / карточки (если есть заказы на автоматизацию)
6. Региональные (РФ, СНГ, удалёнка)
7. Сомнительные / низкое качество — отдельно, не смешивать с основным списком

Дополнительно:
- Для ТОП-20 чатов укажи, есть ли публичная invite-ссылка (t.me/+... или t.me/joinchat/...)
- Оцени риск спама/бана при мониторингу (низкий/средний) — чаты с жёсткой антибот-политикой отметь
- Если знаешь аналоги @freelance, @kwork, @flru — включи
- Источники: открытые каталоги TG-чатов, статьи, Reddit, VC, форумы, GitHub-списки, Telegram-поиск — перечисли откуда взял каждую группу ссылок

Язык ответа: русский. Минимум 80–150 позиций в основном списке, если реально существуют; если меньше — честно напиши сколько нашёл и почему.
Не выдумывай несуществующие @username — если не уверен, пометь «проверить вручную».
```

# Частые вопросы

## Это автоматическая рассылка заказчикам?

**Нет.** RawLead только **находит заказы** и присылает вам уведомление. Писать заказчикам — вы сами, в удобное время. Никакого автоспама.

## Подходит ли для нетехнических специалистов?

Да. Сервис работает для любой ниши: дизайн, иллюстрации, копирайт, SMM, таргет, переводы, разработка, маркетинг — укажите свои теги, система найдёт подходящее.

## Какие источники поддерживаются?

Биржи фриланса (FL.ru, Kwork и другие), сайты-агрегаторы заказов, Telegram-каналы с заявками. База источников постоянно расширяется.

## Нужен ли мой основной аккаунт Telegram?

Для уведомлений — бот пишет в ваш чат, это безопасно. Для мониторинга каналов — отдельная схема, без риска для личного аккаунта. Подробности при регистрации.

## Мне нужен VPS или выделенный сервер?

Нет. Сервис облачный — всё работает без вашего компьютера, 24/7.

## Когда публичный запуск?

Идёт закрытое тестирование на реальных заказах. Оставьте заявку на странице **Контакты** — сообщим первым о старте и условиях раннего доступа.

# Тикеты инцидентов

Навигация по проекту: **[`../README.md`](../README.md)** · твои шаги: **[`../FOR_YOU.md`](../FOR_YOU.md)**

**Зачем:** Lead заводит файл здесь → **Mechanic** чинит код → решение в тикете → **Lead** обновляет `team/TASKS` / `FOR_YOU` при необходимости.

**Цикл**

1. **Lead** создаёт здесь файл `YYYY-MM-DD-кратко.md` по шаблону ниже, статус **открыто**.
2. Ты открываешь **новый чат** → Agent → **`@mechanic`**, прикрепляешь `@docs/problems/…` и при необходимости фрагмент лога (без `.env`).
3. **Mechanic** правит `src/` (и т.д.), закрывает тикет в файле.
4. Ты → **Lead**: «тикет `docs/problems/…` закрыт, нужен ревью / обнови FOR_YOU».

**Имя файла:** `2026-02-12-fl-502-error.md` — дата + короткий slug.

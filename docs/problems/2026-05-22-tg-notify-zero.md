# TG: notify=0 в логе, уведомлений нет

**Статус:** исправлено (логи + ссылки + кнопка) · 2026-05-22

## Симптом

`data/radar.log`: `tg_msg … new=1 notify=0 err=-` — сообщения видит, в бот не шлёт.

## Причина

1. **ИИ вердикт «МИМО»** на болтовню в чатах (frilanc и т.п.) — это штатно, не баг отправки.
2. Ссылки `t.me/c/` собирались с лишним `100` в id (исправлено в `tg_monitor.py`).
3. Пустой `url` ломал inline-кнопку Bot API (исправлено в `telegram_notify.py`).

## Что сделано

- В лог пишется `err=… skip:ai:МИМО` / `skip:filter` / `skip:budget`.
- Проверка отправки: `python src/main.py --telegram-smoke` (бот @FLPARSINGBOT, прокси).

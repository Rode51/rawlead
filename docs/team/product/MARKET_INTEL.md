## 2. АНАТОМИЯ КРАЖИ — ФИЧИ И МЕХАНИКИ

### Что удерживает фрилансеров ежедневно

**1. Скоростное преимущество («быть первым»)**  
Официальный бот FL.ru задерживает уведомление на 2–5 минут. Конкуренты — 10–30 секунд.  
Тезис: «первый отклик получает в 3 раза больше шансов на успех» — создаёт страх пропустить.  
Пользователь проверяет бот непрерывно → daily retention без дополнительных усилий.

**2. Скрытые источники (TG-чаты = рынок без конкуренции)**  
Лидопад и Golubin работают с прямыми заявками в бизнес-чатах, которые не попадают на биржи.  
Клиент пишет «ищу SMM» в чате — там нет 40 откликов с демпингом.  
Для фрилансера это воспринимается как «секретный рынок».

**3. Снятие когнитивной нагрузки (готовый ответ)**  
fl-bot.ru и Freelance PRO генерируют черновик отклика прямо в уведомлении.  
Habit loop: `получил уведомление → кнопка → скопировал → отправил → ждёт ответа` = 2 минуты.  
Это формирует ежедневный ритуал использования продукта.

---

### 3 фичи для немедленного внедрения в RawLead

#### Фича 1 — Скоростной badge «Горячий» (бэкенд уже готов)

В карточке лида — метка времени и дельта «X минут назад».  
Если `NOW() - created_at < 5 минут` → красный badge «Горячий» / «Новое».  
**Бэкенд:** `created_at` в `raw_leads` уже есть. Считается в `/v1/feed`.  
**Дизайн:** один badge-компонент в Lo-Fi UI.  
**Психология:** срочность без алгоритмических изменений.

#### Фича 2 — Черновик отклика с учётом `user_tags` (Direction D, приоритет)

Конкуренты генерируют универсальный шаблон — без знания профиля.  
RawLead знает `user_tags` из Neon → черновик учитывает специализацию.  
**Пример:** «Вы подходите по тегам [Python, FastAPI] — вот как это сформулировать».  
Это то, чего нет ни у одного конкурента в ценовом диапазоне 300–990 ₽/мес.  
**Статус:** уже в roadmap как Direction D (ИИ-агент). Это ядро дифференциации.

#### Фича 3 — Пауза подписки (антихёрн, бэкенд = 1 поле)

Golubin Bot ввёл «приостановить без штрафов» — стало ключевым selling point.  
Без паузы: пользователь отменяет → регистрируется заново через 2 месяца.  
С паузой: замораживает данные, возвращается — и уже привязан.  
**Технически:** поле `subs.paused_until TIMESTAMP` в Neon + крон-джоб проверки.  
**Эффект:** снижение оттока на ~30% (индустриальный benchmark 2026).

## 3. МОНЕТИЗАЦИЯ КОНКУРЕНТОВ

### Механики

| Механика | Кто использует | Суть |
|---|---|---|
| **Trial → auto-renew** | Лидопад, Golubin | 3–7 дней бесплатно, потом списание без подтверждения |
| **Freemium с лимитом** | FBot (1 000 уведомлений) | Hook до привычки, потом платёжка |
| **Ценовые якоря периодов** | fl-bot.ru, Golubin | 1/3/6/12 мес; якорь «73 ₽/день = дешевле кофе» |
| **Апсел premium-фичи** | FBot (автоответы) | Дешёвая база → дорогой апсел |
| **Paywall per category** | Golubin Bot | Каждая ниша = отдельная подписка → x2–x3 ARPU |
| **Custom category за деньги** | Golubin Bot | 5 000 ₽ за добавление своей ниши |

### Churn: что говорят данные 2026

| Метрика | Значение |
|---|---|
| Средний monthly churn, freelance-платформы | **8.5%** (→ 66% годового оттока) |
| Monthly buyer churn (project-based) | 7–10% |
| Платформы с pause/retainer механикой | **< 5%/мес** |
| D90 retention, SaaS-инструменты | 65–75% |
| Пик оттока | **D30–D60** (первая проверка ценности) |
| Возврат при freemium-hook | Лучше, чем после отмены и повторной регистрации |

**Главный вывод по монетизации:**  
Триал, в котором пользователь получил реальный match и черновик отклика **до первого платежа** — сильнее любой скидки. Не «попробуй за 149 ₽», а «получи 3 матча бесплатно, потом реши».

## 4. ТРИ ВАРИАНТА ПИВОТА

### Пивот A — «Лидопад для Digital Pro»
**Прямые заявки из TG + ИИ-скоринг**

**Суть:**  
Парсим TG-чаты с заявками (Telethon уже есть), режем шлак ИИ-модерацией (`ai_score`).  
Лидопад берёт 449 ₽/мес за сырой поток. RawLead берёт **990 ₽/мес за валидированный**.

**Дифференциация:** Лидопад и Golubin не имеют ИИ-скоринга — пользователь получает шум.  
RawLead показывает только заявки с `ai_score ≥ 60` → «меньше уведомлений, но все стоящие».

**Что крадём:** категорийная структура Лидопада + пауза подписки Golubin + trivia-hook FBot.

**Риск:** TG-чаты без явной модерации — серая зона (§0h). Только публичные тематические каналы.

**Оценка усилий:** Средние — Telethon уже парсит TG, нужен источник-каталог публичных бизнес-чатов.

---

### Пивот B — «ИИ-агент с памятью профиля»
**Персональный агент, который знает тебя** ← **РЕКОМЕНДУЕМЫЙ**

**Суть:**  
Конкуренты: универсальный шаблон отклика для всех.  
RawLead: агент знает `user_tags`, историю матчей, рыночную цену по нише → черновик **под профиль**.

**Уникальная механика:**
1. Карточка лида → кнопка «Написать отклик»
2. ИИ генерирует текст с учётом тегов юзера + рыночной цены по данным базы
3. После отклика — фидбэк «принято / отклонено» → агент улучшается
4. Push в TG при новом высоком матче (`final_rank ≥ X`)

**Ценообразование:** 590–990 ₽/мес — выше рынка, обосновано персонализацией.

**Почему это работает:** FL.ru Pro = 1 270 ₽/мес только за доступ к откликам. RawLead за 990 ₽ = подбор + черновик + push. Это выгоднее.

**Риск минимален** — это Direction D из `PRODUCT_VISION.md §0c`. Пивот = ускорение roadmap, не разворот.

**Что крадём:** habit loop fl-bot.ru + Golubin пауза + trivia freemium hook.

---

### Пивот C — «Биржа наоборот»
**Эксклюзивные лиды в одни руки**

**Суть:**  
Все конкуренты продают один заказ всем подписчикам → убивает ценность.  
Топ-10% заказов (`ai_score ≥ 80`) — **не в открытой ленте**, а «ранний доступ» одному подписчику на 4–6 часов.

**Механика paywall:**
- Стандарт: общая лента (300–500 ₽/мес)
- Премиум: «ранний доступ» топ-заказов на 6 часов (990 ₽/мес)
- Эксклюзив: конкретный жирный лид «только тебе» за разовую плату 200–500 ₽/лид

**Почему это деньги:** снимает конкурентное давление — главную боль фрилансера. Аналог: продажа горячих лидов агентствами в B2B. В B2C-агрегаторах для фрилансеров РФ/СНГ **этой механики нет**.

**Что крадём:** логику «закрытых клубов» из премиум TG-каналов + принцип категорийного paywall Golubin.

**Риск:** нужен объём ≥ 200 качественных лидов/день. Запускать только после стабилизации ingest (3i агрегаторы).

## Твои правила (без кода)

1. **Не** копируй `.env` и `.session` в чат, Git, Telegram.
2. **Не** давай Lead править код — только Coder/Mechanic.
3. Команды из docs — **`.venv\Scripts\python.exe`**, не голый `python`.
4. Сломалось → Lead → тикет → Mechanic (не чинить «наугад» в Lead-чате).
5. Новая фича → Lead → `CODER_PROMPT.md` → Coder.
6. Подозрение на утечку бота → BotFather revoke → новый токен в `.env`.
7. **Бэкап** — **перед** большими правками и раз в неделю; после фичи — тоже ([`BACKUP.md`](../ops/BACKUP.md)).

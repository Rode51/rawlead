### Правила безопасности (обязательно)

| Правило | Зачем |
|---------|-------|
| **Нет URL в тексте** | меньше удалений и банов |
| **Ссылка только на картинке** (`@rawlead_bot`) | модераторы реже триггерятся |
| **5–7 чатов/день/аккаунт** | flood-limit TG |
| **30+ мин между постами** | не выглядеть как рассылка |
| **Не acc1/acc2/acc3 радара** | не сжечь инфраструктуру |
| **3 варианта текста** | антидубликат |
| **Стоп при mute/ban** | переключись на резервный аккаунт |

**Честно:** картинка со ссылкой **не даёт 100% защиты** — в некоторых чатах банят любую «рекламу». Тест на 3 чатах в День 1 обязателен.

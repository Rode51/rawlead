## 4. LLM — две модели (env)

| Переменная | Назначение | Модель (пример) |
|------------|------------|-----------------|
| `OPENROUTER_MODEL_SUMMARY` | Задача + 2–3 bullets `ai_reasons` | `deepseek/deepseek-chat` |
| `OPENROUTER_MODEL_VERDICT` | опционально; иначе правила по бюджету/ключам | тот же или отключено |

Промпт: **только** title + snippet `body` (≤600 симв), не full page.

### Оценка стоимости (владелец)

10 000 лидов × ~1 000 токенов ≈ 10M токенов × $0.14/M ≈ **$1.4/день** — **согласен**, если:

- один короткий вызов на лид;
- ingest уже отфильтровал мусор словами;
- категория **не** в LLM.

При 500 лидах/день — копейки.

## 6. Dogfood vs публичная лента

| | Публичная `/lenta/` | Dogfood (владелец) |
|--|---------------------|---------------------|
| Источники | whitelist § P1 | все + старые TG |
| Категории | фильтр пользователя | не режем в боте |
| LLM | тот же pipeline | тот же, бот шлёт всё прошедшее фильтр слов |

API:

- `GET /v1/feed` — public rules + `category`
- Бот — без `category` filter (как сейчас)
- Опционально: `GET /v1/me/feed` для кабинета владельца = полный набор

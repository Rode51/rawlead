# Видение — сайт для фрилансеров (SaaS, позже)

Версия: **0.1** · Lead · 2026-05-22

**Сейчас:** личный радар на ПК (фаза 0–1). **Параллельно:** учебный WordPress **локально** — навык и черновик будущего продукта.  
**Не путать** с «пультом радара» ([`TZ_WP.md`](TZ_WP.md)) — тот подключается к уже работающему движку.

---

## 1. Идея продукта

**Сервис для фрилансеров с подпиской:**

- мониторинг **бирж** (FL, Kwork, …) и **Telegram-чатов**;
- у каждого свои **направления** и **ключевые слова**;
- **ИИ-разбор** «стоит ли откликаться» + черновики;
- уведомления в Telegram (или в кабинете на сайте).

**Не в MVP SaaS:** автоспам в ЛС, авто-отклик на биржах без человека.

---

## 2. Роли частей системы

```
[Сайт WordPress]          ← лендинг, тарифы, оплата, кабинет (настройки)
        ↕ REST/API
[Радар Python на VPS]     ← парсинг, Telethon, ИИ, очереди
        ↕
[Postgres Neon]           ← пользователи, фильтры, лиды, подписки
        ↕
[Telegram Bot]            ← пуши юзеру
```

| Слой | Технология | Зачем |
|------|------------|--------|
| Витрина + биллинг | **WordPress** (+ WooCommerce / MemberPress позже) | SEO, привычный стек заказов, оформление |
| Движок | **Python** (текущий `src/`) | Парсинг, ИИ, TG — не на shared-хостинге |
| Данные | **Neon** | Уже есть для личного радара; позже мультитенант |

**Вывод:** WordPress — **сайт и бизнес-оболочка**; мониторинг — **не** «плагин на Timeweb за 200 ₽».

## 3. Этапы (не смешивать)

| Этап | Что | Кто |
|------|-----|-----|
| **A. Сейчас** | Радар для себя: FL/Kwork → бот → Neon | Coder + ты |
| **B. Учёба** | Локальный WP: лендинг-скелет, дизайн, формы | **Ты** + [`../ops/WP_LOCAL_SKELETON.md`](../ops/WP_LOCAL_SKELETON.md) |
| **C. TG** | Чаты в радаре | Coder после API_ID |
| **D. Пульт v1** | WP на хосте читает **твоего** радара (настройки/лиды) | Coder + [`TZ_WP.md`](TZ_WP.md) |
| **E. SaaS** | Много пользователей, подписка, свои фильтры в БД | Отдельное ТЗ после A+C доказали ценность |

Детали SaaS-бэклога: [`TZ.md`](TZ.md) §10.

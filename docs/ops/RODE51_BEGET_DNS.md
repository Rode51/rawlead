## Шаг 1. Купить или добавить домен

1. Зайди на [beget.com](https://beget.com) → войди в аккаунт.
2. Меню **«Домены»** (или **«Регистрация доменов»**).
3. В поиске введи **`rode51.ru`**.
4. Если свободен — **«Зарегистрировать»** (~199–400 ₽/год), оплати.
5. Если уже куплен у Beget — он появится в списке **«Мои домены»**.

> Домен куплен не на Beget? См. [шаг 6](#шаг-6-домен-не-на-beget).

---

## Шаг 2. Открыть DNS домена

1. **«Домены»** → **«Мои домены»**.
2. Клик по **`rode51.ru`**.
3. Вкладка **«DNS»** / **«Управление DNS»** / **«Подзона»** (название может чуть отличаться).

Должен быть список записей (иногда пустой или с заглушками Beget).

---

## Шаг 3. Добавить A-запись на VPS

Нужно **две** записи (корень и www):

### Запись 1 — сам сайт `rode51.ru`

| Поле | Значение |
|------|----------|
| Тип | **A** |
| Имя / Subdomain | **`@`** или **пусто** (корень домена) |
| Значение / IP | **`62.113.103.231`** |
| TTL | по умолчанию |

### Запись 2 — `www.rode51.ru`

| Поле | Значение |
|------|----------|
| Тип | **A** |
| Имя | **`www`** |
| Значение | **`62.113.103.231`** |

**Сохранить.** Старые A/CNAME на `@` и `www`, которые ведут на парковку Beget, — **удали или замени**, иначе будет конфликт.

## Шаг 5. Написать Lead «DNS готов»

Когда `ping` показывает IP VPS — напиши в чат:

> DNS готов, rode51.ru

Lead / Coder:

- nginx vhost `rode51.ru`
- `certbot --nginx -d rode51.ru -d www.rode51.ru`
- в коде портфолио URL `https://rode51.ru`
- redeploy static

## Шаг 6. Домен не на Beget

Если `rode51.ru` куплен на Reg.ru / Timeweb / др.:

**Вариант A — DNS оставить у регистратора**  
Там же добавь A: `@` и `www` → `62.113.103.231`.

**Вариант B — перенести DNS на Beget**  
В панели регистратора смени NS на Beget (смотри в карточке домена на Beget → «DNS-серверы», обычно `ns1.beget.com`, `ns2.beget.com`).

## Частые ошибки

| Проблема | Решение |
|----------|---------|
| ping не тот IP | Подожди TTL · проверь, что A именно на `62.113.103.231` |
| Две A на `@` | Оставь одну |
| Сайт Beget «заглушка» | Убери forwarding/парковку в панели домена |
| HTTPS не работает | Сначала HTTP + DNS, потом certbot (делает Lead) |

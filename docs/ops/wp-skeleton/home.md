# Радар заказов для фрилансеров

**FL.ru, Kwork и Telegram-чаты** — в одном потоке. ИИ отсеивает мусор и подсказывает, стоит ли писать заказчику.

[Смотреть тарифы](#) · [Как это работает](#)

---

## Для кого

- Разработчики и верстальщики на **Cursor / WordPress**
- Те, кто устал вручную мониторить биржи и чаты
- Фрилансеры, которые **сами** пишут в ЛС (без автоспама)

---

## Что получаете

| | |
|---|---|
| 🔍 | Новые заказы с бирж и из выбранных TG-чатов |
| 🎯 | Свои **ключевые слова** и стоп-слова |
| 🤖 | Разбор ИИ: брать / сомнительно / пропустить |
| 📱 | Уведомления в **Telegram** |

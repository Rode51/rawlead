## Риски (из исследования)

| Риск | Действие |
|------|----------|
| Спам-блок при массовых ЛС | Не откликаться с +66; лимит 15–20 холодных ЛС/сутки на рабочем аккаунте |
| Private invite `t.me/+…` | Нужен `tg_join`; после join — id в `SOURCES_POOLS` |
| Шум / дубли с биржами | `FILTER_WIDE=1` + стопы в [`FILTERS.md`](FILTERS.md); позже дедуп TG↔FL |
| Публичные api_id 2496 | Временно; свои ключи — когда получится |

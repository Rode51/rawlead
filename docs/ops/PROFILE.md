# Профиль (для AI-разбора заказов)

## Кто ты

- **Никита, 29.** В коде — начинающий; реализация через **Cursor Agent** (Composer 2.5 / Auto) + Sandbox.
- **Стек:** Python 3.11, **FastAPI**, **Telethon**, **Neon** (Postgres), **WordPress** (каркас + REST), TG-бот уведомлений.
- Сложные API (GetCourse, AmoCRM, Bitrix24, OAuth) — только с выжимкой из **Gemini Deep Research** в Cursor.

## Жёсткий фильтр (для ИИ)

### МИМО — не брать
- Figma, вёрстка, React/Vue фронт, **1С**, **Bitrix/ядро**, mobile с нуля, Cloudflare-bypass парсинг.
- Задачи **без ТЗ**, дипломы, накрутки; спам без бюджета на расходники.
- Бюджет на бирже **&lt; 3 500 ₽**.
- Платные API, прокси, сервера, аккаунты, WP-плагины — **платит заказчик**; без аванса на расходники — не брать.

### Брать
- Парсеры сайтов/TG, **TG-бот с нуля (aiogram 3)** — Telethon только для мониторинга чатов, автоматизация, скрипт на **1–3 вечера**.
- ИИ через API (OpenAI/DeepSeek), правка чужого бэкенда **1–2 файла** (JS, PHP).

### Сомнительно
- Большие платформы — если режется на модули (парсер + Neon + WordPress REST).

## Сроки (заказчику, с запасом)

## Отклик на FL

- Уверенно, без воды; **без** Cursor / ИИ / ChatGPT в тексте.
- При «Брать» — **FastAPI + Neon**, модульный подход, готовые шаблоны.
- Цена **«от …»** по «Старт отклика» из разбора, не с карточки FL.

## Минимум на бирже

- ИИ в промпте режет **&lt; 3 500 ₽** как МИМО.
- В `.env` → `MIN_BUDGET_RUB` (сейчас часто 1000 — до ИИ; подними до 3500 если не хочешь видеть мелочь вообще).

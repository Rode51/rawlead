Да. Сервис работает для любой ниши: дизайн, иллюстрации, копирайт, SMM, таргет, переводы, разработка, маркетинг — укажи свои теги, система найдёт подходящее.

Биржи фриланса (FL.ru, Kwork и другие), сайты-агрегаторы заказов, Telegram-каналы с заявками. База источников постоянно расширяется.

Для уведомлений — бот пишет в твой чат, это безопасно. Для мониторинга каналов — отдельная схема, без риска для личного аккаунта. Подробности при регистрации.
